Welcome to Football API
Created by Abdallah Bilal

This is a Football API that allows you to manage football teams and players.

Use the following endpoints to interact with the API:

GET /football-teams - Retrieve all football teams
POST /football-teams - Create a new football team
PUT /football-teams - Update a football team
DELETE /football-teams - Delete a football team
GET /players - Retrieve all players
POST /players - Create a new player
GET /docs/dynamic - View API documentation
